# ChatGPT Batch Translator - Portable Build

## Tao file EXE

Tren may tao ban build, chay:

```powershell
build_portable.bat
```

Sau khi build xong, app nam tai:

```text
dist\ChatGPT Batch Translator\
```

Nen gui ca thu muc `ChatGPT Batch Translator` cho nguoi dung, khong chi gui rieng file `.exe`.

## Cach nguoi dung chay

1. Giai nen thu muc portable.
2. Mo `ChatGPT Batch Translator.exe`.
3. Chon thu muc anh goc va thu muc luu anh VN.
4. Bam `Chay batch moi`.
5. Lan dau app se mo browser de dang nhap ChatGPT.

## Khong dong goi cac thu muc nay cua may ban

- `chatgpt_auto_profile`
- `images`
- `images_vn`
- `Kvant`
- `Kvant VN`
- `app_settings.json`
- `progress.csv`

Moi nguoi dung nen co profile ChatGPT va file cau hinh rieng tren may cua ho.
