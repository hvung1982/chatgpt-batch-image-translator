# Security Policy

## Tiếng Việt

Nếu bạn phát hiện vấn đề bảo mật, vui lòng không đăng công khai session, cookie, token, ảnh riêng tư, hoặc dữ liệu cá nhân trong GitHub Issues.

Hãy tạo issue chỉ mô tả ngắn gọn loại vấn đề và cách liên hệ, hoặc liên hệ trực tiếp với maintainer nếu có kênh riêng được công bố trong repo.

## Dữ liệu nhạy cảm cần tránh chia sẻ

- Thư mục `chatgpt_auto_profile` hoặc profile Playwright trên macOS.
- Cookie/session đăng nhập ChatGPT.
- File `app_settings.json`.
- Ảnh gốc, ảnh output, hoặc nội dung sách/truyện không có quyền chia sẻ.
- Log có chứa đường dẫn cá nhân, tên tài khoản, hoặc nội dung riêng tư.

## English

If you find a security issue, please do not publicly post sessions, cookies, tokens, private images, or personal data in GitHub Issues.

Open an issue with a short description of the issue type and a way to contact you, or contact the maintainer directly if a private channel is listed in the repository.

## Sensitive data not to share

- The `chatgpt_auto_profile` folder or the Playwright profile on macOS.
- ChatGPT login cookies/sessions.
- The `app_settings.json` file.
- Input/output images or book/comic content you do not have permission to share.
- Logs containing personal paths, account names, or private content.
